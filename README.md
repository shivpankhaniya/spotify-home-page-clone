# 🎵 Spotify Clone

A responsive **Spotify Clone** built using **HTML** and **CSS**. This project recreates the Spotify Web Player interface, including the sidebar, navigation bar, playlists, music player, and footer controls.

---

## 🚀 Features

- 🎧 Spotify-inspired UI
- 📂 Sidebar Navigation
- 🔍 Search Bar
- 📚 Playlist Cards
- 🎵 Music Player
- ⏯️ Playback Controls
- 📊 Progress Bar
- 🔊 Volume Control
- 📱 Responsive Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts (Montserrat)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/spotify-clone.git
```

2. Open the project folder

```bash
cd spotify-clone
```

3. Open `index.html` in your browser.
